# ☁️ CloudNotes

CloudNotes is a cloud-based note-taking application built using **React**, **Django**, and **SQLite**, and deployed on **Google Cloud Platform (GCP)**. The project demonstrates key cloud computing concepts such as virtual machines, cloud storage, DNS mapping, firewall configuration, and full-stack application deployment.

# project live url http://myycloudnotes.mooo.com:8000/

## Features

* Create Notes
* View Notes
* Search Notes
* Delete Notes
* Responsive User Interface

## Technologies Used

### Frontend

* React.js

### Backend

* Django
* Django REST Framework

### Database

* SQLite

### Cloud Services

* Google Compute Engine (VM)
* Google Cloud Storage
* Public IP Address
* Firewall Rules
* FreeDNS Domain Mapping

## Deployment

### Backend

* Created a GCP Virtual Machine
* Configured Python Virtual Environment
* Installed Django and DRF
* Ran database migrations
* Hosted Django server

### Frontend

* Built React application
* Uploaded production build to VM
* Served frontend through Django

## Cloud Computing Concepts Demonstrated

* Infrastructure as a Service (IaaS)
* Virtual Machines
* Cloud Storage
* DNS Configuration
* Firewall Management
* REST APIs
* Web Hosting
* Full-Stack Deployment

## Future Enhancements
